HTML・CSS サンプルコード

自分がこのブログで使用するために作成したCSSは以下の通りです。

CSS

    .code1 { /* 短めのコード表示用　*/
      position: relative;
      width: 100%;
      margin: 2.0em 0;
      padding: 0.5em 0;
      font-size: 16px;
      font-weight: bold;
      letter-spacing: 0.10em;
      line-height: 1.8em;
      color: #FFF;
      background-color: #555;
    }

    .code1__title {
      position: absolute;
      text-align: center;
      font-size: 13px;
      width: 20%;
      background-color: #F4A460;
    }

タイトルとコードをそれぞれ分けています。

タイトルはコードがどのディレクトリに属するか、

また何の言語を使用しているかを表示します。

短いコード用のデザインなので文字の大きさ・字間・行間を大きめに取っています。


文字の大きさ　=> font-size: ~~~;

字間　=> letter-spacing: ~~~;

行間　=> line-height: ~~;


※クラス名は"CSS設計の教科書"という本を参考にしています。
